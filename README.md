# NopCommerce UI Automation Testing

Это проект по автоматизации тестирования UI онлайн магазина NopCommerce. Он создан с целью практической отработки навыков по автоматизированному тестированию UI, с упором на построение масштабируемой архитектуры UI авто-тестов,
применение современных практик автоматизации и приближение структуры тестов к реальным
production-проектам.

---

## О проекте

Проект нацелен на автоматизацию тестирования UI онлайн магазина NopCommerce и покрывает smoke тесты и критические e2e сценарии для следующих функциональных блоков:

- **Authentication** – регистрация и логин пользователей
- **Home Page** – тесты для главной страницы
- **Product Page** – тесты для страницы товара
- **Cart Page** – тесты для страницы корзины
- **Checkout Page** – тесты для страницы оформления заказа

---

## Технологический стек

В проекте используются следующие инструменты и технологии:

- **Python 3.13**
- **Pytest** — фреймворк для написания тестов и управления ими (фикстуры, плагины, маркеры)
- **Playwright** — современная библиотека для автоматизации тестирования UI
- **Allure** — инструмент для генерации отчетов (шаги, теги, критичность)
- **Faker** — библиотека для генерации тестовых данных
- **Playwright tracer** — инструмент для отслеживания и анализа выполнения тестов

---

## Технические особенности реализации

Проект построен с использованием паттернов Page Object Model, Page Component Model и Page Element Model.

Используется иерархия:
- Page — страницы приложения
- Block — логические блоки страницы
- Component — логические UI компоненты
- Element — атомарные UI элементы
